# toughwlan-pypy

这是一个专供ToughWLAN使用的基础Docker镜像。封装了pypy来代替python，实现性能提升。

## Docker镜像名

    talincode/toughwlan-pypy
    
## 包含模块

    supervisor
    Mako
    Beaker
    MarkupSafe
    PyYAML
    Twisted
    treq
    tablib
    cyclone
    six
    autobahn
    pycrypto
    pyOpenSSL>=0.14
    service_identity
    MySQL-python
    SQLAlchemy
    beanstalkc
