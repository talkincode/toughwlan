# TOUGHWLAN介绍

TOUGHWLAN 是一个开源的WLAN运营管理系统，采用于AGPL许可协议发布。

TOUGHWLAN的目标是提供了一个完整的无线运营解决方案，包含PORTAL认证，配置管理，并提供开放的接口支持第三方业务扩展。

## TOUGHWLAN 快速指南

    $ easy_install toughcli

    $ toughcli wlan --install
